# 丛林法则Game聊天体验版 — 测试操作手册

> 后端地址：https://game-chat-28fa.onrender.com  
> 体验码有效期：2026-05-21（到期后在微信后台重新设置）

---

## 一、当前状态确认（开发者操作）

### 1.1 先解决"无法连接网络"——配置合法域名

**这是体验版必须做的一步，不同于开发者工具（开发者工具可以跳过域名校验，体验版不行）。**

1. 打开 https://mp.weixin.qq.com，用管理员微信号登录
2. 左侧菜单：**开发** → **开发管理** → **开发设置**
3. 下拉找到 **服务器域名** → **request合法域名**
4. 点「修改」，添加：
   ```
   https://game-chat-28fa.onrender.com
   ```
5. 点「保存并提交」，等待生效（通常立即生效）

### 1.2 重新上传体验版（域名修改后必须重传）

1. 打开**微信开发者工具**，确认左上角项目为 `game_chat/miniprogram`
2. 按 `Ctrl+B` 重新编译（或点工具栏「编译」）
3. 点工具栏右上角 **上传**
4. 版本号填 `0.1.0-beta`，备注填 `配置合法域名，修复网络连接`
5. 点确认上传

### 1.3 在后台设为体验版

1. 打开 https://mp.weixin.qq.com → **版本管理**
2. 在「开发版本」找到刚刚上传的版本
3. 点右侧 **···** → **选为体验版**
4. 确认后体验版码自动更新（二维码不变，内容会更新）

## 二、添加测试成员

1. mp.weixin.qq.com → **成员管理** → **体验成员**
2. 点「添加体验成员」
3. 输入测试者**微信号**（不是昵称），或让测试者扫管理员生成的邀请码
4. 最多可添加 15 人

## 三、测试者操作流程

### 3.1 进入小程序

1. 微信扫描体验版二维码（或开发者发来的链接）
2. 首次进入需授权：点「允许」
3. 看到主页面：**丛林法则Game聊天助手**

### 3.2 功能测试清单

#### ✅ 测试1：创建对话
1. 点首页「+ 新建对话」按钮
2. 输入对象名字（如：小红）
3. 选择目标（恋爱 / 朋友 / 职场）
4. 可选填备注（认识多久、背景等）
5. 点「开始」
6. **预期结果**：跳转到聊天页，显示对话标题

#### ✅ 测试2：消息分析
1. 在聊天页，在底部输入框粘贴/输入对方发来的消息
2. 点「分析」按钮
3. **预期结果**：约 3-10 秒后（首次请求因 Render 冷启动可能需 30 秒），显示 AI 分析结果
4. 分析内容包括：意图解读、建议回复、注意事项

> ⚠️ **冷启动说明**：Render 免费版 15 分钟无请求后会休眠。第一次请求需等待约 20-30 秒，之后响应正常（1-3秒）。这是正常现象。

#### ✅ 测试3：查看/切换历史对话
1. 返回首页
2. 可以看到之前创建的对话列表
3. 点击任意对话，可继续分析

#### ✅ 测试4：删除对话
1. 在首页，长按某个对话
2. 或进入对话后点右上角「...」→ 删除
3. **预期结果**：对话从列表中消失

### 3.3 反馈收集

测试完成后，请告知开发者以下信息：

| 项目 | 描述 |
|------|------|
| 首次加载时间 | 点分析到出结果大约几秒？ |
| 结果质量 | AI 分析是否准确/有用？ |
| 操作流程 | 有没有哪个步骤不清楚或报错？ |
| 建议 | 希望增加什么功能？ |

## 四、常见问题排查

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 「无法连接到网络」 | 合法域名未添加，或体验版未更新 | 按第一章步骤操作后重新上传 |
| 「域名未加入白名单」 | 同上 | 同上 |
| 首次分析等很久 | Render 冷启动（正常） | 等待 30 秒，后续正常 |
| 分析结果为空 | DeepSeek API Key 问题 | 检查 Render 环境变量 DEEPSEEK_API_KEY |
| 页面空白/加载圈不停 | 小程序版本过旧 | 退出小程序重进，或删除重新扫码 |
| 扫码提示"体验版不可用" | 未添加为体验成员 | 管理员在后台添加微信号 |
